Michael G. DeGroote Institute for Infectious Disease Research, David Braley Center for Antibiotic Discovery

Supervisor: <a href='https://mcarthurbioinformatics.ca/'>Dr. Andrew G. McArthur</a>
  
I'm interested in using natural langauge processing for antimicrobial resistance surveillance. I've used text mining techniques (e.g., text classificaiton, named entity recogntion, relation extraction) to extract antibiotic information (e.g., minimum inhibitory concentration) from biomedical literature. My current work involves developing a multi-agentic framework for biocuration.
